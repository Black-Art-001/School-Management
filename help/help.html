üîß Troubleshooting

Common Issues and Solutions:

Issue: Database won't open
Solution: Check if file is in use by another program, ensure it has .db extension

Issue: Slow performance with large data
Solution: Use the built-in database optimization tool (Vacuum)

Issue: Buttons are disabled
Solution: Ensure a database is currently open

Issue: Data not saving
Solution: Check disk space and file permissions
