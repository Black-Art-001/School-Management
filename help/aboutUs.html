About This Project

The School Management System is a specialized database management application designed specifically for school administration. Built from scratch with C++ and Qt, it provides a comprehensive solution for managing student data, teacher information, and academic records through an intuitive desktop interface.

Core Purpose: To provide schools with a reliable, offline-first database management system that simplifies data organization while maintaining complete control over sensitive educational records.

Architecture Details

The application follows a clean architecture with:

  • MainWindow: Primary UI controller handling all user interactions
  • DataBase Class: SQLite wrapper for all database operations
  • AddTable Dialog: Custom table creation interface
  • Signal/Slot Architecture: Event-driven design for responsive UI
  • QSqlQueryModel: Data model for table view display
